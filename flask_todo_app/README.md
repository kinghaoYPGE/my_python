| -------- | -------------- | -------------- |

### 需求功能：

- [ ] 用户注册


    **注：不仅可以站内注册，也可使用第三方注册(如：github，qq，wechat等)**

- [ ] 用户登陆

- [ ] 用户退出

- [ ] 查询所有事项

- [ ] 增加待办事项

- [ ] 修改代办事项内容

- [ ] 标记完成代办事项

- [ ] 用户和权限管理

  **注：只有root权限才可以进入管理页面增删改用户数据**
